## What slice 2.1 added

| Piece | File | Role |
|---|---|---|
| Jukebox | `DuneIIRendering/Audio/Jukebox.swift` | `@MainActor` `AVMIDIPlayer` wrapper. `play(named:)` loads XMI → SMF via `Formats.Xmi.Song.Track.toStandardMidiFile()` → plays via `AVMIDIPlayer(data:soundBankURL:)`. `nil` soundBank falls back to the system DLS. `stop()` + `isPlaying` for lifecycle. |
| Voice | `DuneIIRendering/Audio/Voice.swift` | `@MainActor` VOC player on `AVAudioEngine` + `AVAudioPlayerNode`. `Voice.makeBuffer(for:)` converts u8-PCM → float32 with `(byte - 128) / 128.0`. |
| WSA loader | `AssetLoader.loadWsa(named:)` | Returns `WsaFrames { frames: [CGImage], width, height }`; honours the WSA's embedded palette when present. |
| XMI loader | `AssetLoader.loadXmiAsSMF(named:)` | Returns SMF-ready `Data?` for `AVMIDIPlayer`. |
| VOC loader | `AssetLoader.loadVoc(named:)` | Returns `Formats.Voc.Sound?`. |
| Intro scene | `DuneIIRendering/Scene/IntroScene.swift` | Swap-texture WSA playback at 15 fps (≈ vanilla 70/5 Hz). Click / end → `.mainMenu`. |
| Boot wiring | `GameController` | Owns a `Jukebox` + optional `Voice`. `start()` routes to `.intro` when `INTRO.WSA` exists, else `.mainMenu`. `route(to: .mainMenu)` kicks off `DUNE0.XMI`. |

Six new install-gated tests (`AudioIntroTests.swift`) pin:

1. `INTRO.WSA` decodes into 320×200 frames.
2. `DUNE0.XMI` produces an SMF whose first four bytes are `MThd`.
3. A real `.VOC` from the install decodes with non-zero rate + non-empty samples.
4. `Jukebox.play(named:)` against a real XMI succeeds.
5. `Voice.makeBuffer` pins the u8→float remap (0 → -1.0, 128 → 0.0).
6. `Voice.makeBuffer` rejects empty / zero-rate input.

## Why no bundled SoundFont

`AVMIDIPlayer(soundBankURL: nil)` falls back to `DLSMusicDevice` — Apple's GM-compatible sound bank that ships with every macOS install. That's enough to make `DUNE0.XMI` audible. Shipping a custom `.sf2` would need a license audit (GeneralUser GS is CC-0 and would fit, but adds 30 MB to the tree). User can drop any `.sf2` / `.dls` at `~/Library/Audio/Sounds/Banks/` and the system will pick it up; a future config knob on `Jukebox` can make this explicit.
